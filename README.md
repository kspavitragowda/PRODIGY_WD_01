# PRODIGY_WD_01
Prodigy InfoTech Web Development Internship - Task 1

Responsive Interactive Navigation Menu


Objective

Create an interactive navigation menu that changes color or style when scrolled or when hovering over a menu item. The navigation menu remains fixed at the top of the page and is visible while navigating through different sections.

Features

- Fixed navigation bar
- Navigation menu changes background color when scrolling
- Hover effects on menu items
- Smooth scrolling between sections
- Responsive design for desktop, tablet, and mobile devices
- Built using HTML, Cascading Style Sheets (CSS), and JavaScript

Technologies Used

- HTML
- Cascading Style Sheets (CSS)
- JavaScript

How It Works

- The navigation bar is fixed at the top of the page.
- JavaScript detects the user's scroll position.
- When the page is scrolled, a new Cascading Style Sheets class is applied to the navigation bar, changing its appearance.
- Menu items change color when hovered over using Cascading Style Sheets hover effects.
